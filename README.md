## 入选标准

1. 我觉得内容还不错的博客（目前是这样，你觉得有更好的标准可以说出来）。
2. 不追潮流，注重沉淀。
3. 关注于解决问题，专注于更透彻的理解 Web 开发。
4. 欢迎 PR
